## Group Work and Individual Contributions

This project was developed as a group coursework submission for the Internet Application Development (IAD) course under the Bachelor of Technology (BTech) program at Ebenezer Higher Institute of Science and Technology (EHIST), Bamenda.

Each team member contributed to specific parts of the application based on assigned responsibilities.

### Documented Contributions

Ntoh Batex Bafika  
JavaScript logic, API integration, and application flow

Core responsibilities included:
- Designing and implementing asynchronous logic using async and await
- Fetching real-time weather data using the Fetch API
- Integrating the OpenWeatherMap API
- Implementing browser Geolocation API handling and permission flow
- Designing fallback logic when location access is denied
- Implementing structured error handling using try and catch
- Dynamically updating the user interface without page refresh
- Ensuring reusable and maintainable function design
